# tamagotchi-jam
Funny game for joke

Flow is now: **Select Pet → Egg (tap to hatch) → Pet life**.
After **Reset** or **Death**, the game returns to pet selection for a new life.

https://lukaszpeciak91-sys.github.io/tamagotchi-jam/
